# Smart Student Onboarding Agent

## Problem Statement
Student onboarding is fragmented across departments leading to confusion and delays.

## Solution
AI-powered onboarding assistant that provides:
- Personalized onboarding checklist
- AI-based next action recommendation
- Context-aware chat assistant
- Admin analytics dashboard

## Tech Stack
Frontend: React + Tailwind
Backend: Django + DRF
AI: Groq API
Database: SQLite

## Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
cd frontend
npm install
npm run dev
